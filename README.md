# Calorie AR Vision

An iOS application that uses live video stream and LLMs to analyze dishes and display calorie breakups in an augmented reality format.

## Project Structure
- `/ios`: SwiftUI application with ARKit integration.
- `/backend`: FastAPI server for dish analysis and nutrition data.

## Tech Stack
- **Frontend**: Swift, SwiftUI, ARKit, Combine.
- **Backend**: Python, FastAPI, VLM (GPT-4o/Gemini Pro Vision).
- **API**: Nutritionix/Edamam for calorie data.
